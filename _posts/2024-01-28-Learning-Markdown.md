# 2. Text Styles

### **BOLD** and *ITALIC*:
>##### _italic_:   
>>**Syntax**(`_ _ or * *`)  
➡️ **Example**(`*this is italic* | _this also_`)  
➡️ **Output**(*this is italic* | _this also_)   
>            
> #### bold:   
>>**Syntax**(`__ __ or ** **`)  
➡️ **Example**(`**This is bold** | __This also__`)  
➡️**Output**(**This is bold** | __This also__)  
>            
##### Both bold and italic:  
>>**Syntax**(`___ ___ or *** ***`)  
➡️ **Example**(`***This is bold and italic*** | ___This also___`)  
➡️**Output**(***This is bold and italic*** | ___This also___)
>      
> ##### Bold and _nested italic_:
>>**Syntax**(`**_ _**`)  
➡️ **Example**(`**This is _italic_ inside bold**`)  
➡️**Output**(**This is _italic_ inside bold**)      

### **<sub>Subscript</sub> and <sup>Superscript</sup>** 
>
>##### <sub>Subscript</sub>
>>**Syntax**(`<sub> </sub>`)  
➡️ **Example**(`This is a <sub>SubScript</sub> text`)  ➡️**Output**(This is a <sub>SubScript</sub> text)
>
>##### <sup>Superscript</sup>
>>**Syntax**(`<sup> </sup>`)  
➡️ **Example**(`This is a <sup>SuperScript</sup> text`)  ➡️**Output**(This is a <sup>SuperScript</sup> text)




# 3. 🔗Links:

> ### 1. Inline link:
>> **Syntax**   
>>>`[wrap the __link text__ in brackets]( wrap the __link__ in parentheses)`
>>        
>> **Example** 
>>>`[Pirikli's _GitHub_](https://github.com/pirikli)`    
>>    
>> **OUtput** ➡️➡️[**Pirikli**'s _GitHub_](https://github.com/pirikli)    
 
> ### 2. Reference link:          
>> **Syntax**    
>>          <blockquote> [link text][__refrence__ to another place inside doc]      
>>           [__refrence__ to another place inside doc]: **_link_**-to-outside-websites  </blockquote>  
>> 
>> **Example**    
>>   <blockquote>  [learn turki] [ refrence-to-link ]     
>>                 [ refrence-to-link ]: https://mirze.ir  </blockquote>      
>>          
>>**Outpute** ➡️➡️   [**_learn turki_**][refrence-to-link] 

[refrence-to-link]: https://mirze.ir

# 5. <img src="https://cdn-icons-png.flaticon.com/128/852/852907.png" alt="blocs" width="50" height="auto">Blockquote
### Turn a line or a paragraph to a blockquote.
   - **Syntax:**   
     Put a Greater than sign ( > ) at the begining.
   - **Example:**   
         > This line is a blockquote.  
   - **Output:**   
     > <font color="green">This line is an example of a blocquote.</font>
     
     > I am turning this paragraph to a blocquote with just one Greater than sign (>).
       me aware that the lines mustn't have spaces in between   
       The lines must continuous not separated.  
### Turn separated lines or more than one paragraph to blockquote.   
   - **Syntax:**   
     Put a Greater than sign ( > ) at the begining and between each line.
   - **Example:**   
         > This is the first line of the first paragraph.   
           This is second line.   
           That's enough.  
         >   
         > This is Second paragraph.       
           lets put them together in one Blocquote.      
     
   - **Output:**
      > This is a paragraph    
           with lines without    
           space in between.    
         >     
         > This is Second paragraph.         
           lets put them together in one Blocquote   
           by just addin > in the space.  

### Indented blockquotes.   
   - **Syntax:**   
      Use Double Greater than sign ( >> )
   - **Example:**   
         > This is outer blociquoe   
         >> This is inner blockquote:   
         >   >> This line is the most inner one.
         >>  >> You can go deeper inside...   
         >   >>  >>  deeper and deeper!   
         >>   >>  >> again   
         >>   >>  >>> agian and again   
         >>   >>  >>>> You can also use this inline syntax: <mark><🅱️lockquote></mark> the Deepest one❗<mark><🅱️lockquote></mark> 
   - **Output:**
      > This is outer blociquoe   
         >> This is inner blockquote:   
      >   >> This line is the most inner one.
         >>  >> You can go deeper inside...   
      >   >>  >>  deeper and deeper!   
      >>   >>  >> again   
      >>   >>  >>> agian and again   
      >>   >>  >>>> You can also use this inline syntax: <blockquote> the Deepest one❗ <blockquote>
